About This Project

This is a fully automated landing page deployed via GitHub Actions, Terraform, and Nginx on AWS EC2. Every push to the main branch triggers deployment, demonstrating a complete CI/CD pipeline.

Features

- Automated deployment to EC2 on push
- Nginx serving static content
- HTTPS enabled with self-signed certificate
- Responsive and colorful design
